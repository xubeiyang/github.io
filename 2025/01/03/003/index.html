二、创建云端服务

1. 在GitHub上创建新项目

  • 登录GitHub：访问 GitHub 并登录您的账号。
  • 创建新仓库：
    • 点击页面右上角的 + 按钮，选择 New repository。
    • 在 Repository name 中输入 username.github.io（将 username 替换为您的GitHub用户名）。
    • 选择 Public（公开），然后点击 Create repository。

2. 生成SSH Keys

  • 打开Git Bash，输入以下命令生成SSH密钥：

  • 复制SSH公钥：
    • 进入 C:\Users\用户名\.ssh 文件夹。
    • 用记事本打开 id_rsa.pub 文件，复制全部内容。

3. 在GitHub中添加SSH Key

  • 进入GitHub设置：
    • 点击右上角头像，选择 Settings。
    • 在左侧菜单中找到 SSH and GPG keys。
  • 添加SSH Key：
    • 点击 New SSH key。
    • 在 Title 中起一个名字（如 My Blog Key）。
    • 在 Key 中粘贴之前复制的 id_rsa.pub 文件内容。
    • 点击 Add SSH key 完成添加。
